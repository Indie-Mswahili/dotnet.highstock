dotnet.highstock
================

본 프로젝트는 DotNet.Highcharts(https://dotnethighcharts.codeplex.com/)를 확장 한 것 입니다.
익히 알고 있듯이 웹 페이지에 차트를 그려 주는 highchart는 많은 인기를 얻고 있는 중 입니다.

허나, 본인과 같이 닷넷 c# 프로그래머 인 경우 dotnet.highstock이 필요 하게 되었고
이를 기존에 open 되어 있는 dotnet.highcharts를 이용하여 DotNet.Highstock을 코딩 하게 되었습니다.

본 프로젝트는 2014/12에 시작 되었으며, 테스트를 완벽히 수행 하지 못한 상태 입니다.
지속적으로 보완 할 계획이오니 많은 활용 바랍니다.
